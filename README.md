## About

Covers testing strategies for Spring Boot applications using JUnit and Mockito. This assignment demonstrates how to write unit tests for isolated components and integration tests for end-to-end verification.

## Key Concepts

- JUnit 5 annotations (`@Test`, `@BeforeEach`, `@DisplayName`)
- Mockito mocking and stubbing (`@Mock`, `@InjectMocks`, `when/thenReturn`)
- Integration testing with `@SpringBootTest`
- Test-driven development (TDD) principles
